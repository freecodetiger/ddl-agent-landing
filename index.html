为什么选择 DDL Agent

核心特色

数据完全本地化

所有日程和待办都存在你的手机上，离线可用，不依赖服务端存储。后端只是 AI 推理网关 — 你的数据从不上传。

交互

自然语言输入

说"明天下午三点开会"就能创建日程。支持中文口语表达、模糊时间、重复规则。

安全

Propose-Apply 机制

AI 生成操作提案，你确认、修改或取消后才写入。AI 永远不会直接修改你的数据。

输入

语音识别

内置阿里云实时语音识别，说话比打字更快。支持边走边记录。

费用

自带 API Key

在设置中填入你的 DeepSeek Key，按量付费，费用和用量完全由你掌控。

视觉

深浅双主题

浅色和深色两套主题，跟随系统自动切换，夜间使用不刺眼。
